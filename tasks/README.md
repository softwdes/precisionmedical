## How to use these task files

### For Antigravity (or AI agents)
1. Read the entire phase file before starting
2. Execute tasks in order
3. Each task has a "Definition of done" — meet it before moving on
4. Mark tasks complete in your tracking system
5. Demo each phase to the project lead before proceeding

### For human developers
1. Use these as a backlog
2. Each task has an estimated time
3. Tasks within a phase can be parallelized if no dependencies
4. Phases must complete sequentially
